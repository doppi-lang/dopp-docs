### Doppi Language Notes

<Note title="Function Declaration">
**Funksiya:**  
`.doppi` tilida funksiyalar `funksiya` kalit so'zi bilan e'lon qilinadi. Masalan:
```js
funksiya funksiyaNomi(parametr) {
    // Funksiya ichidagi kod
}
```

**Funksiya chaqiruvlari:**  
Funksiyani chaqirish uchun uning nomini yozib, kerakli parametrlarni uzatish kerak:
```js
funksiyaNomi(parametr);
```

**Taqqoslash:**  
- `.doppi`da `funksiya` kalit so'zi orqali funksiyalar e'lon qilinadi, bu Java, TypeScript, Python kabi tillarda ham mavjud.
</Note>
