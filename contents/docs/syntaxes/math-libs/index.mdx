# Matematik Funksiyalar

Doppi tili matematik hisob-kitoblarni osonlashtirish uchun bir nechta foydali funksiyalarni taqdim etadi. Quyida mavjud bo'lgan funksiyalar va ularning ishlatilishining qisqacha tavsifi keltirilgan.

## 1. Kosinus (`cosinus`)
`matematik.cosinus(x)` - Berilgan burchakning kosinusini hisoblaydi. Burchak radianlarda beriladi.

**Sintaksis**:
```js
matematik.cosinus(x)
```

**Misol**:
```js
matematik.cosinus(30) // Kosinus(30°) ning qiymatini qaytaradi
```

---

## 2. Tangens (`tangens`)
`matematik.tangens(x)` - Berilgan burchakning tangensini hisoblaydi. Burchak radianlarda beriladi.

**Sintaksis**:
```js
matematik.tangens(x)
```

**Misol**:
```js
matematik.tangens(60) // Tangens(60°) ning qiymatini qaytaradi
```

---

## 3. Kvadrat ildiz (`ildiz`)
`matematik.ildiz(x)` - Berilgan sonning kvadrat ildizini hisoblaydi.

**Sintaksis**:
```js
matematik.ildiz(x)
```

**Misol**:
```js
matematik.ildiz(16) // 16 sonining kvadrat ildizini qaytaradi (4)
```

---

## 4. Daraja (`daraja`)
`matematik.daraja(x, y)` - Birinchi sonni ikkinchi son darajasiga ko'taradi (x^y).

**Sintaksis**:
```js
matematik.daraja(x, y)
```

**Misol**:
```js
matematik.daraja(2, 3) // 2 ning 3-darajasi (8)
```

---

## 5. Modul (`module`)
`matematik.module(x, y)` - Birinchi sonning ikkinchi songa bo'linishidan qolgan qoldiqni qaytaradi.

**Sintaksis**:
```js
matematik.module(x, y)
```

**Misol**:
```js
matematik.module(10, 3) // 10 ning 3 ga bo'linishidan qolgan qoldiq (1)
```

---

## 6. Maximum (`max`)
`matematik.max(x, y, ...)` - Berilgan sonlar orasidagi eng katta qiymatni qaytaradi.

**Sintaksis**:
```js
matematik.max(x, y, ...)
```

**Misol**:
```js
matematik.max(1, 4, 7, 3) // 7 sonini qaytaradi
```

---

## 7. Minimum (`min`)
`matematik.min(x, y, ...)` - Berilgan sonlar orasidagi eng kichik qiymatni qaytaradi.

**Sintaksis**:
```js
matematik.min(x, y, ...)
```

**Misol**:
```js
matematik.min(1, 4, 7, 3) // 1 sonini qaytaradi
```

---

## 8. Giperbolik sinus (`sinh`)
`matematik.sinh(x)` - Berilgan sonning giperbolik sinusini hisoblaydi.

**Sintaksis**:
```js
matematik.sinh(x)
```

**Misol**:
```js
matematik.sinh(1) // 1 ning giperbolik sinusini hisoblaydi
```

---

# Xulosa

Doppi tili matematik hisob-kitoblarni bajarish uchun juda qulay va sodda sintaksisga ega funksiyalarni taqdim etadi. Ushbu funksiyalar yordamida, kvadrat ildizdan tortib, giperbolik trigonometrik funksiyalargacha turli xil operatsiyalarni bajarish mumkin.

Funksiyalarni chaqirish uchun `matematik.functionName()` formatidan foydalanish mumkin. Barcha funksiyalar oddiy va intuitiv sintaksis bilan ishlaydi, bu esa dasturlashni osonlashtiradi.
