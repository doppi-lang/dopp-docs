### Doppi Tilda Shartli Operatorlar

Doppi dasturlash tilida shartli operatorlar yordamida turli holatlarni boshqarish mumkin. Bu tilda faqat `agar` va `aks` mavjud, `aks agar` ishlatilmaydi. Kod oqimini boshqarishda `agar` qismidagi tugallanmasida `aks` mavjud bo'lsa, oxiriga nuqta-vergul (`;`) qo'yilmaydi, faqat `aks` qismida nuqta-vergul ishlatiladi.

---

## Shartli Operatorlar Sintaksisi

```js
agar (shart) {
    // Shart bajarilganda ishlatiladigan kod
} aks {
    // Shart bajarilmasa ishlatiladigan kod
};
```
> Agar inkor amalini ishlatmoqchi bo'lsangiz emas() kalit so'zini ishlatamiz. misol uchun **(rost && yolgon || emas(rost && yolgon ))**

**Misol:**

```js
yarat num := 10;

agar (num == 10) {
    chiqar("10 ni qaytardi")
} aks {
    chiqar("20 ni qaytardi");
};

// Shartlardan tashqari umumiy holat
chiqar("30 ni qaytar");
```

---

### Ichma-ich Shartlar

Doppi tilda ichma-ich joylashgan shartlar bilan murakkab tekshirishlarni amalga oshirish mumkin.

```js
agar (num > 0) {
    chiqar("Musbat son");
    agar (num % 2 == 0) {
        chiqar("Juft son");
    } aks {
        chiqar("Toq son");
    };
} aks {
    chiqar("Manfiy yoki nol");
};
```

---

## Kengaytirilgan Misol

Murakkab shartli konstruktsiyani faqat `agar` va `aks` yordamida yaratish mumkin. Quyida baho baholash uchun dastur ko'rsatilgan:

```js
yarat baho := 85;

agar (baho >= 90) {
    chiqar("A darajasi");
} aks {
    agar (baho >= 80) {
        chiqar("B darajasi");
    } aks {
        chiqar("C darajasi");
    };
};
```

### Doppi Tilda Shartli Operatorlar

<Note title="Shartli Operatorlar">
Doppi tildagi asosiy farqlar:
- `agar` - Shartni ifodalash uchun ishlatiladi.
- `aks` - Shart bajarilmaganda bajariladigan kodni yozish uchun ishlatiladi.
- `aks` tugagandan keyin nuqta-vergul (`;`) qo'shiladi.
</Note>

### Xulosa

Doppi tilda shartli operatorlar sodda va tushunarli tuzilgan. Har qanday murakkab shartni faqat `agar` va `aks` yordamida ifodalash mumkin. Bu dasturlash tili strukturasi ixcham va o'qilishi oson bo'lishiga katta yordam beradi.
